# MB Signal Search Tool V5 版本更新说明

## V5版本新增功能

### 1. 模糊搜索
- **下划线模糊匹配**：搜索 `mpa_state` 和 `mpastate` 结果相同
- **空格模糊匹配**：搜索 `park apa` 自动忽略空格
- **枚举值模糊搜索**：搜索枚举值名称也能匹配

### 2. 回车键搜索
- 在搜索框输入关键词后，直接按 **Enter** 键即可触发搜索
- 无需点击搜索按钮

### 3. Excel上传更换信号列表
- 点击工具栏 **"📊 更换信号列表"** 按钮
- 上传本地Excel文件（.xlsx 或 .xls）
- 自动解析以下Sheet：
  - Mapping_MMT
  - Mapping_IPC
  - Mapping_RoV
  - Mapping_IPC_DC
  - Mapping_SEM
  - MappingTemplate
  - P6FHAL_DataType_IPC

### 4. 下载最新信号列表
- 点击工具栏 **"⬇️ 下载最新信号列表"**
- 可快速打开SharePoint下载页面获取最新Excel

### 5. 位长度显示
- 新增"位长度"列，显示信号的位数：
  - `bool` → 1位
  - `uint8_t` → 8位
  - `uint16_t` → 16位
  - `uint32_t` / `float` → 32位
  - `uint64_t` / `double` → 64位

### 6. 物理意义显示
- 没有枚举值的信号会显示物理意义（来自Excel的Description字段）

## 使用方法

1. **打开工具**：双击 `Signal_Search_Tool_V5.html`
2. **搜索信号**：输入关键词，按Enter键
3. **更换数据**：点击"更换信号列表"上传新Excel
4. **下载最新**：点击"下载最新信号列表"
